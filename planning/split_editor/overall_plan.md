# エディタ上下分割 計画

## 目的

現在の編集画面（左：Edit カメラ）を上下に分割し、
- **上パネル**：基図形を描く Edit パネル（既存機能をそのまま移植）
- **下パネル**：レプリカを配置・操作する Placement パネル（新規）

## 現在のレイアウト

```
┌─────────────┬─────────────┬─────────────┐
│             │             │             │
│  Edit       │  Result     │  params     │
│  (left)     │  (right)    │  panel      │
│             │             │  (egui)     │
└─────────────┴─────────────┴─────────────┘
```

## 目標レイアウト

```
┌──────────┬──────────┬──────────┐
│          │          │          │
│  Edit    │  Result  │  params  │
│  (上)    │          │  panel   │
├──────────┤          │  (egui)  │
│          │          │          │
│ Placement│          │          │
│  (下)    │          │          │
└──────────┴──────────┴──────────┘
```

## フェーズ一覧

| Phase | 内容 | 概要 |
|-------|------|------|
| 1 | レイアウト分割 | PlacementCamera 追加、ビューポートを上下分割 |
| 2 | Placement 描画 | 基図形を薄い色で表示、レプリカを色付きで表示 |
| 3 | レプリカ操作 | クリック選択・ドラッグで移動/スケール/回転 |

## 受け入れ条件

- [ ] 上パネルで線を描く既存機能が正常に動作する
- [ ] 下パネルに基図形が薄い色（またはアルファ低め）で表示される
- [ ] 下パネルにレプリカが各色で表示される
- [ ] レプリカをクリックすると枠線が表示される
- [ ] 枠線内ドラッグで移動できる
- [ ] 枠のドラッグ（エッジ）でスケールできる
- [ ] 枠外ドラッグで回転できる
- [ ] Result パネルが正しく表示される（回帰なし）
