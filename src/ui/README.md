# `ui`

## 責務

**プレゼンテーション層**。egui のパネル・レイアウト、[各 `Camera` の `Viewport` 更新](../viewport_bridge.rs)、種／配置／結果キャンバスでの入力。**ドメインの規則**は `ui` だけでは完結しない。[状態変更は `app` 経由](../app/mod.rs)に寄せ、[プラットフォーム具象](../platform/mod.rs)への **直接の依存は持たない**（I/O は `ports` と起動時の束ね側に集約）。

## サブツリー

| ディレクトリ | 責務（一文） |
|------------------|----------------|
| [`frame`](frame/mod.rs) | ワイド／ナロー、パラメータ、クローム類、結果上の深度オーバーレイなどキャンバス外枠・パネル |
| [`canvas`](canvas/mod.rs) | Seed・Placement・Result のワールド系キャンバスと格子ガイド |
| [`feedback`](feedback/mod.rs) | トースト表示（文言や出す判断は [`app`](../app/mod.rs) 寄りのリソースイベント） |

[`UiPlugin`](mod.rs) は上記プラグイン群を組み込む親。
