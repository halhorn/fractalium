# Phase 2: Placement パネル描画

## 目的

Placement パネルに以下を描画する：
1. 基図形を薄い色（アルファ 0.25 程度）で表示
2. 各レプリカに基図形を適用した図形を対応色で表示

## Tasks

### Task 1: 基図形のゴースト描画

`placement.rs` に Gizmo グループ `PlacementGizmos` を定義し、
`PlacementCamera` のレンダーレイヤに割り当てる。

基図形の線を `GHOST_COLOR`（例: `Color::srgba(0.9, 0.9, 1.0, 0.25)`）で描画する。

### Task 2: レプリカの描画

`edit.rs` の `draw_replicas` と同様に、
各レプリカに基図形を適用した線を対応色で Placement ギズモに描画する。
アルファは 1.0（フル）でよい。

## 受け入れ条件

- [ ] Placement パネルに基図形が薄い色で表示される
- [ ] 基図形が未設定（lines が空）の場合は何も描画されない
- [ ] レプリカが設定されているとき、各レプリカが対応色で描画される
- [ ] Edit パネルの描画に影響がない（レイヤが独立している）
