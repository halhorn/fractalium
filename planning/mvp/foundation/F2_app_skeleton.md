# F2: アプリ骨組み（レイアウト）

## 目的

操作部・結果表示部・パラメータ編集パネルの 3 領域を画面上に明確に配置し、ウィンドウリサイズに追従する骨組みを作る。中身（実際の図形描画・編集 UI）は空でよく、後続の F3 / Features がそれぞれの領域に実装を流し込めるようにする。

## レイアウト方針

横長デスクトップを想定し、以下の 3 領域構成とする。

```
+---------------------------------------------+-------------+
|                       |                     |             |
|   操作キャンバス      |  結果表示キャンバス  |  egui       |
|  (基本図形 + 複製枠)  |  (フラクタル描画)    |  サイドパネル|
|                       |                     |  (パラメータ)|
|                       |                     |             |
+---------------------------------------------+-------------+
```

- 右端: egui `SidePanel::right`（幅 ~300 px、パラメータ編集用、F2 ではラベルのみ表示）
- 左残り: 操作キャンバスと結果表示キャンバスを左右等分（各カメラに viewport 矩形を設定）
- 操作キャンバスと結果表示キャンバスは独立した 2D カメラ。各キャンバスは [-1, 1] 座標を持つが、座標系の正規化自体は F3 で実装する（F2 では単純にウィンドウ座標で動作確認）

## 計画概要

| ID | 内容 |
|----|------|
| T1 | `EguiPlugin` の追加と最小動作確認（"Fractalium" タイトルバー / 右サイドパネルにラベル） |
| T2 | 操作キャンバス用カメラと結果表示用カメラの 2 つを `Startup` で生成 |

### T2-T3: 2 カメラ + viewport

- 構造: `EditCamera` / `ResultCamera` の 2 つの marker component を定義
- 各 `Camera2d` を marker component と一緒に spawn
- `Camera::viewport` フィールドに `Viewport { physical_position, physical_size, depth: 0.0..1.0 }` を毎フレーム再計算して設定
- viewport 矩形計算: `(window_width - panel_width) / 2` ずつを左右のキャンバスに割り当て
- パネル幅は固定値（例: 300）から開始。後続でユーザー調整可能化を検討

### T4: ラベル仮置き

- 各カメラのビュー内に、`Text2d` / 単色矩形 / または egui の中央パネル風表示で "Edit" / "Result" のラベルを置く
- 実装の楽さを優先し、Bevy の `Text2d` を 1 つずつ各カメラ視野に配置する想定（カメラごとに `RenderLayers` を分けて、片方にしか映らないようにする）

[Question]
`RenderLayers` を使った視野の分離は Bevy 0.18 でどう書くか実装時に確認する。代替として、操作キャンバスと結果キャンバスで spawn するエンティティをまったく別系統にし、各カメラの位置を空間的に離すことでも分離できる。
[/Question]

[Draft]
F2 ではシンプルに「異なるワールド座標位置に各キャンバスのコンテンツを配置し、各カメラが自分の領域だけを見るようにする」方式で実装する。具体的には:
- 操作キャンバスの内容は world (0, 0) 周辺に配置
- 結果キャンバスの内容は world (10000, 0) 周辺に配置
- 各カメラの `Transform.translation` をそれぞれに合わせて配置

これは MVP 段階の暫定解。コンテンツが増えたら `RenderLayers` 化を検討する。

### T5: リサイズ追従

- viewport 計算を `Update` システムで毎フレーム行うことで、ウィンドウサイズ変化に追従する
- パネル幅も egui が描画した実幅を取得（`ctx.available_rect()` 等）して反映するのが理想だが、MVP では固定幅 300 で開始
