# 目的

Ctrl を押している間、編集画面にグリッドを表示し、線の端点がグリッド上の点にのみスナップするようにする。直交グリッドと等角グリッドを重ねた「結合スナップグリッド」により、四角形・三角形・正三角形をきれいに描けるようにする。

# 計画概要

## Task 1: スナップ関数の実装

`src/draw.rs` に以下を追加する。

- **直交グリッド**: スナップ間隔 = `2.0 / 8.0 = 0.25`（8等分 → 2, 4, 8分割に対応）
- **等角グリッド**: 水平間隔 = `2.0 / 6.0 ≈ 0.333`、垂直間隔 = 水平間隔 × `sin(60°)`（6等分 → 3, 6分割・正三角形に対応）
- `snap_to_grid(pos: Vec2) -> Vec2`: 全スナップ点の中で最近傍の点を返す

## Task 2: Ctrl キー検知とスナップ適用

`handle_drag_input` および `draw_lines` で Ctrl キーを検知し、`snap_to_grid()` を適用する。

- ドラッグ開始時の始点もグリッドにスナップする
- ドラッグ解放時の終点もグリッドにスナップする
- プレビュー描画でもスナップ後の位置を使う
- Ctrl + Shift の同時押しでグリッドスナップ + 45°スナップを両方適用

## Task 3: グリッドの可視化

Ctrl を押している間、編集キャンバス上にグリッドの点（ドット）を Gizmos で描画する。

- 直交グリッド点: 淡いドット（小さい円）
- 等角グリッド点: 淡いドット（同サイズ）
- カーソルに最も近いスナップ点: 強調表示（大きい円や異なる色）

# 計画詳細

## 座標系

キャンバスは正規化座標 [-1, 1] × [-1, 1]（幅・高さともに 2.0）。

```
直交グリッド:
  x スナップ点 = -1.0, -0.75, -0.5, -0.25, 0.0, 0.25, 0.5, 0.75, 1.0
  y スナップ点 = 同上（合計 9×9 = 81点）

等角グリッド（六方格子）:
  水平間隔 d = 2.0 / 6.0 = 1/3
  垂直間隔 h = d × sqrt(3) / 2 ≈ 0.289
  偶数行: x = -1 + n × d
  奇数行: x = -1 + (n + 0.5) × d  ← 半周期ずれる
```

## スナップ優先度

全スナップ点から最近傍を選ぶ（距離最小）。同距離の場合は等角グリッド点を優先。

## 色定数（追加）

```rust
const GRID_DOT_COLOR: Color = Color::srgba(0.6, 0.7, 1.0, 0.4);     // グリッド点: 薄青
const GRID_SNAP_COLOR: Color = Color::srgba(0.4, 1.0, 0.6, 0.8);    // 最近傍点: 緑
const CTRL_SNAP_COLOR: Color = Color::srgb(0.4, 0.8, 1.0);          // Ctrl スナップ時のプレビュー線: 水色
```

# 受け入れ条件

- [ ] Ctrl を押すとグリッドのドットが表示される
- [ ] Ctrl を押した状態でドラッグすると、始点・終点ともにグリッド点にスナップされる
- [ ] 直交グリッドで縦・横・斜め 45° の線を正確に引ける（四角形が作れる）
- [ ] 等角グリッドで 60° の線を正確に引ける（正三角形が作れる）
- [ ] Ctrl + Shift の同時押しも正常動作する
- [ ] Ctrl を離すとグリッドが非表示になる
- [ ] カーソルに最も近いスナップ点が強調表示される
