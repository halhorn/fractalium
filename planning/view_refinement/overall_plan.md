# ビュー整理 計画

## 目的

3 つのキャンバス（Base Shape / Placement / Result）とパラメータパネルのレイアウト・見栄え・操作感を整える。
あわせて、Base Shape 上で線分そのものを選択・移動・端点編集できるようにし、編集体験を「描いて終わり」から「描いて整える」へ拡張する。

## 現在のレイアウト

```
┌──────────┬──────────┬──────────┐
│ BaseShape│          │          │
│  (1/2 H) │  Result  │  Params  │
├──────────┤          │ (300px)  │
│ Placement│          │          │
│  (1/2 H) │          │          │
└──────────┴──────────┴──────────┘
```

## 目標レイアウト

```
┌──────────┬──────────────────┬────────┐
│ BaseShape│                  │        │
│ □ 正方形 │                  │ Params │
├──────────┤      Result      │ (やや  │
│ Placement│  （タイトル無し）│  狭い） │
│ □ 正方形 │                  │ ［▶]   │
├──────────┤                  │ で最小 │
│  余白    │                  │  化可  │
└──────────┴──────────────────┴────────┘
```

- 左ペインは Base Shape → Placement の順に「上から詰めて」スタック。各パネルは縦横比固定（正方形）。
- 各パネルには egui の枠（パラメータパネルと同じ枠線＋背景色）でくくり、内側にパディング。
- ヘッダ部にタイトル（大きめ）と操作ボタン（Clear lines / + Add replica など）を集約。
- 描画領域の背景は Result キャンバスと同じ色（`ClearColor` の `srgb(0.08, 0.08, 0.10)`）。
- Base Shape / Placement の描画上の外枠（白い枠線スプライト）はもっと暗い色で目立たなくする。
- パラメータパネルは右端にトグルボタンで最小化／復帰できる。
- Result のタイトルラベルは削除。

## フェーズ一覧

| Phase | 内容 | 概要 |
|-------|------|------|
| 1 | レイアウト・スタイル整備 | 左ペイン egui パネル化、上詰め＋縦横比固定、ヘッダ集約、外枠暗色化、Params 幅調整＋最小化、Result タイトル削除 |
| 2 | ドラッグ確定タイムラグ解消 | マウスを離してから値が確定するまでの遅延を調査し解消 |
| 3 | Base Shape の線編集 | 線分のクリック選択・本体ドラッグ移動・端点ドラッグ移動・Ctrl/Shift スナップ対応 |

詳細は各 Phase のファイルを参照。

- [Phase 1: レイアウト・スタイル整備](phase1_layout.md)
- [Phase 2: ドラッグ確定タイムラグ解消](phase2_drag_lag.md)
- [Phase 3: Base Shape の線編集](phase3_base_shape_edit.md)

## 影響範囲（概要）

- `src/main.rs` … 外枠スプライト色
- `src/ui.rs` … パラメータパネル幅・最小化、左ペイン egui パネル化、ヘッダ集約、Result タイトル削除、ビューポート計算
- `src/edit.rs` … Base Shape の線選択・移動・端点ドラッグ、ドラッグ確定遅延の調査
- `src/placement.rs` … ドラッグ確定遅延の調査・修正
- `src/state.rs` … 編集状態（選択中線分・端点ドラッグ）追加、パラメータパネルの折りたたみ状態

## 受け入れ条件（全体）

- [ ] Phase 1 が完了している
- [ ] Phase 2 が完了している
- [ ] Phase 3 が完了している
- [ ] 既存機能（線描画／レプリカ操作／フラクタル描画）に回帰がない

## 非対象

- パラメータパネルの内容変更（既存スライダ・コラプスドヘッダの構成は維持）
- Result キャンバスのレイアウト変更（ズーム挙動含む）
